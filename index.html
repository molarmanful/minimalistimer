Help

Press the spacebar to start/stop the timer. Start inspection with ctrl + spacebar. Click the scramble to change it.

Press the display to start/stop the timer. Tap the scramble to change it. Change device orientation to landscape to show statistics.

MinimalisTimer aims to give its users the best experience possible. If you see a bug or have a suggestion, you can submit a Github issue or post to the Speedsolving forum thread.

Don't forget to try out some experimental stuff:

  • 3SetTimer: A 3x3 subset trainer built on an older version of MinimalisTimer.
  • Material Edition: A reskin of MinimalisTimer using Materialize.

Enjoy!
